# esBG-Reporting Project Setup

## Server Configuration

### Laravel Backend - Nginx Configuration
- **Port**: 89 (NOT 8000)
- **Server name**: esbg-reporting.localhost
- **Root directory**: /var/www/esBG-reporting/server/public
- **PHP handler**: PHP-FPM socket at /run/php/php8.4-fpm.sock
- **Config location**: /etc/nginx/sites-available/esbg-reporting.conf

```nginx
server {
    listen 89;
    server_name esbg-reporting.localhost;

    root /var/www/esBG-reporting/server/public;
    index index.php;

    # Logs
    access_log /var/log/nginx/esbg-reporting.access.log;
    error_log /var/log/nginx/esbg-reporting.error.log;

    # Gzip compression
    gzip on;
    gzip_types text/plain text/css text/xml text/javascript application/x-javascript application/xml+rss;

    # Security headers
    add_header X-Frame-Options "SAMEORIGIN" always;
    add_header X-Content-Type-Options "nosniff" always;

    # Static files - cache them
    location ~* \.(jpg|jpeg|png|gif|ico|css|js|svg|woff|woff2|ttf|eot)$ {
        expires 365d;
        add_header Cache-Control "public, immutable";
    }

    # Block access to sensitive files
    location ~ /\.(htaccess|env|git) {
        deny all;
    }

    # Route all requests through Laravel
    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }

    # PHP handler - pass to PHP-FPM socket
    location ~ \.php$ {
        fastcgi_pass unix:/run/php/php8.4-fpm.sock;
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $realpath_root$fastcgi_script_name;
        include fastcgi_params;
        
        # Additional headers for Sanctum
        fastcgi_param HTTP_ACCEPT_LANGUAGE $http_accept_language;
    }
}
```

## Important Notes
1. Laravel is accessed through Nginx on port 89, NOT port 8000
2. API calls from frontend use the Ngrok URL to reach the backend through the Nginx tunnel
3. When ngrok URL changes, update both `.env` files:
   - `/var/www/esBG-reporting/client-frontend/.env`
   - `/var/www/esBG-reporting/server/.env`
4. Ensure Nginx and PHP-FPM are running
5. **CRITICAL**: Ngrok MUST tunnel to port 89 (not port 80 or 8000): `ngrok http 89`

## Common Issues & Fixes

### 502 Bad Gateway through Ngrok
- **Cause**: Ngrok tunneling to wrong port (default is 80)
- **Fix**: Kill ngrok and restart with correct port: `pkill -f ngrok && ngrok http 89`

### Laravel temp directory errors
- **Cause**: Storage and bootstrap/cache directories owned by root, not www-data
- **Fix**: `chown -R www-data:www-data /var/www/esBG-reporting/server/storage /var/www/esBG-reporting/server/bootstrap/cache`
